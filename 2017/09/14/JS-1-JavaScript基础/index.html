JS-1.JavaScript基础

  • JavaScript
  • 2017-09-14

一，什么是JavaScript以及组成

1.概念

JavaScript是一种基于对象和事件的弱类型（松散类型）语言，是一种解释性语言，js运行在用户端
1995年发布，最初适用于验证表单，开发者是布兰登艾奇

2.JavaScript的组成

a.ECMAscript 语法核心（关键字，保留字，流程控制，变量…）
b.DOM 文档对象模型（Document Object Model）
c.BOM 浏览器对象模型（Browser Object Model）

二，JavaScript的引入方式

1，行内式：直接将js代码写在标签内部
2,内部式：使用script标签，直接写在HTML文档中
3，外部式：外部引入一个js文件

（1）.JS 引入的时候一般放在HTML 所有内容之后
（2）外部式元素内写代码块不会生效

三，JavaScript的输出方式

每个能独立完成一个功能的代码以分号结束 这就是一个代码块

1，document.write()；

向文档中输入内容，不会覆盖原文档的内容，可以识别标签

2，console（控制台输出）(可以直接在浏览器的控制台输入代码)

a，console.log();一般用于调试代码
b，console.dir();打印目录
c，console.clear();清除
d，console.error();抛出一个异常

3，alert();弹出一个带有一条指定信息的消息（警示框）
4，innerHTML/innerText 获取或者向指定元素内添加内容 (innerHTML是可以识别标签)

使用这两个元方法的是需要元素的，如果两个方法一起用的话 后面的会覆盖前面的
会将元素原有的内容覆盖。

四，JavaScript的命名规则

1.严格区分大小写
2.命名时名称可以出现字母、数字、下划线、$ ,但是不能数字开头，也不能纯数字
3.不能包含关键字和保留字。
关键字：var number
除了关键字 top name 也尽量不使用。
4.推荐驼峰命名法：有多个有意义的单词组成名称的时候，第一个单词的首字母小写，其余的单词首字母写
5.匈牙利命名：就是根据数据类型单词的的首字符作为前缀
aLi 多个 array
oUl 单个 object

五，变量

变量：可以看成一个存储数据的容器，用关键字var来声明

六，JavaScript的五种基础数据类型

两种引用数据类型：function object

1，数字 number

整数，小数（浮点数），NaN（not a number）；
将其他类型转换成数字类型：Number();

强制转换：
parseInt();整数
parseFloat();小数
返回值只有两种：数字或者NaN

隐式转换：将数组转换成数字类型时，先进行toString（）方法，将数组转换成string类型，然后再转换成数字类型
true=1，false=0，null=0，undefined=NaN

isNaN();返回值是false代表是数字，返回值是true代表不是数字
parseFloat();小数

2，字符串 string

加引号的
将其他类型转换成字符串类型：String();

3，布尔 boolean： false（假） true（真）

将其他类型转换成字符串类型：Boolean();
在js中除了 0、“ ”、null、undefined、NaN其余都是真，[]空数组会先转成字符串“”再转成布尔类型false，{}空对象转成字符串是“[object Object]”

！表示取反，非
机制：先将后面的值调用Boolean()方法转为布尔类型后取反 返回值true 或者false

4，空（空对象指针） null
5，未定义 undefined

null和undefined的区别：
两个在js中都表示“无”；null表示“没有对象”，预留位置，之后再添加；undefined是本应该有值，但是却没有。

6，检测数据类型 typeof

null的数据类型是object

####六，操作符
一个=号 ：赋值
两个==号：比较值
三个===号：全等，先比较类型，再比较值
NaN不等于任何值，包括它本身

1.算数运算符

运算优先级：++写在前面，先自增再赋值，++写在后面，先赋值再自增

2.赋值运算符

3.比较和逻辑运算符


!==不全等于
&&：先看第一个数是真还是假，如果是真，直接返回第二个值，如果是假，直接返回第一个值
||：先看第一个数是真还是假，如果是真，直接返回第一个值，如果是假，直接返回第二个值
4.条件运算符（三目运算符）
